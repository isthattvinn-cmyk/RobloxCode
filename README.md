# TextAI Roblox Studio

TextAI Roblox Studio is a client-side website for drafting Roblox and Luau systems with a memory-first workflow.

## What It Does

- Keeps chat sessions in browser storage so your project history survives refreshes
- Stores project memory such as the game goal, rules to preserve, and important systems
- Lets you update the current code draft instead of wiping it every time
- Saves version snapshots so you can restore an older working script
- Includes quick prompt launchers for combat, datastore, admin, and quest systems
